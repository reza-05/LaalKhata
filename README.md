# LaalKhata

LaalKhata is a personal finance app built with Flutter, designed to make managing everyday money simple and organized.

Instead of being just another expense tracker, the goal of LaalKhata is to provide a complete personal finance experience where users can manage expenses, multiple money sources, budgets, loans, projects, and financial insights—all in one place.

The project is currently under active development, and new features are being added continuously.

## Features (Work in Progress)

* Secure authentication
* Local PIN & biometric lock
* Multiple money sources (Cash, bKash, Bank, Cards, etc.)
* Expense & income tracking
* Budget management
* Financial summaries
* Loan & due tracking
* Project-based expense management
* Offline-first experience with cloud sync

More features will be added as development progresses.

## Tech Stack

* Flutter
* Dart
* Supabase

## Project Status

🚧 This project is still in development.

The current focus is on building a clean, reliable, and user-friendly experience before adding more advanced features.

## Vision

The long-term goal of LaalKhata is to become a modern digital finance companion that helps people understand and manage their money without making finance feel complicated.
